# Téléchargements Jeannot

Ce dépôt public contient uniquement les versions bêta signées de Jeannot. Le code source de l’application reste privé.

Pour installer la version Android actuelle, ouvrez [Jeannot 1.0.0 bêta 1](https://github.com/damien-schneider/jeannot-downloads/releases/tag/v1.0.0-beta.1) depuis votre téléphone, téléchargez le fichier APK et autorisez temporairement l’installation si Android le demande.

Les mises à jour publiques passeront ensuite par Google Play et l’App Store.

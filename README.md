# Téléchargements Jeannot

Ce dépôt public contient uniquement les versions bêta signées de Jeannot. Le code source de l’application reste privé.

Pour installer la version Android actuelle, ouvrez la [dernière version bêta](https://github.com/damien-schneider/jeannot-downloads/releases/latest) depuis votre téléphone, téléchargez le fichier APK et autorisez temporairement l’installation si Android le demande.

Les mises à jour publiques passeront ensuite par Google Play et l’App Store.
